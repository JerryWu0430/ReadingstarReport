Deployment Manual

Step-by-step guide for deploying ReadingStar

Prerequisites

  • Node.js (v14 or higher)
  • Python 3.8+
  • MongoDB
  • Git

Installation Steps

1. Clone the Repository

git clone https://github.com/username/readingstar.git
cd readingstar

2. Backend Setup

cd backend
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
pip install -r requirements.txt

3. Frontend Setup

cd frontend
npm install

4. Environment Configuration

Create and configure the following environment files:

# backend/.env
DATABASE_URL=mongodb://localhost:27017/readingstar
JWT_SECRET=your_jwt_secret
API_KEY=your_api_key

# frontend/.env
REACT_APP_API_URL=http://localhost:5000

5. Database Setup

cd backend
python manage.py migrate
python manage.py createsuperuser

6. Running the Application

# Terminal 1 (Backend)
cd backend
python manage.py runserver

# Terminal 2 (Frontend)
cd frontend
npm start
